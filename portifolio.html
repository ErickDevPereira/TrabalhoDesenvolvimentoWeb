Compact Media Searcher

Compact Media Searcher foi uma REST API que criei. O objetivo da API é capturar o nome de um título/nome de mídia (música, filme, livro ou jogo digital) junto com o nome do autor de várias APIs na web de forma assíncrona, seguida de um processamento dos dados para filtrar os títulos e autores que mais se aproximam daquele pesquisado pelo usuário por meio de um algorítimo chamado TF-IDF. Ou seja, a API é uma search engine que retorna dados sobre mídias digitais que mais estão próximas daquilo que o usuário pesquisou. Utilizei Python, Flask e Numpy para a implementação da API.

API de dados locais

Esta API recebe latitude e longitude e retorna dados de clima e radiação iônicas do local requerido. Além disso, o usuário pode fazer um comentário a respeito de um ponto do planeta e ler comentários de outras pessoas sobre um determinado ponto. Esta api consome dados de outras apis e faz um merging desses dados e os processa para a devida apresentação de dados limpos e organizados. Usei Python, Pandas, Numpy e Flask para a implementação da API.

Site de análise de gostos de filmes

Site full stack, criado em Python e Flask (com HTML estático como response, sem API), com objetivo de permitir o usuário acessar dados de filmes, ver comentários sobre determinado filme, comentar, e dar like ou deslike em comentários. Além disso, o usuário pode registrar um filme em sua conta, daí, uma apresentação de estatísticas sobre os filmes e gostos desse usuário são apresentadas para o mesmo.
